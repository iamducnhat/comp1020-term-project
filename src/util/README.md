# Utility Package

## Các class cần tạo:
- `ActionHistory.java` - Undo/redo dùng Stack (LIFO)
- `SortUtil.java` - Sorting O(n log n) + Binary Search O(log n)

## Data Structures:
- **Stack**: push/pop cho undo/redo
- **Sorting**: sắp xếp order theo thời gian/giá, customer theo points
- **Binary Search**: tìm customer theo phone trong sorted list
