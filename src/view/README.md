# View Layer (Java Swing GUI)

## Các class cần tạo:
- `MainFrame.java` - Cửa sổ chính, chứa các tab
- `OrderPanel.java` - Giao diện quản lý đơn hàng
- `CustomerPanel.java` - Giao diện quản lý khách hàng
- `InventoryPanel.java` - Giao diện quản lý kho
- `VoucherPanel.java` - Giao diện quản lý voucher

## Gợi ý:
- Dùng JTabbedPane cho navigation giữa các panel
- Dùng JTable cho hiển thị danh sách
- Kết nối với Controller qua action listeners
