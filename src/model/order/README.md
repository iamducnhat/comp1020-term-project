# Model Layer - Order Package

## Các class cần tạo:
- `Order.java` - Đơn hàng chứa danh sách Beverage
- `OrderStatus.java` - Enum (PENDING, PROCESSING, COMPLETED, CANCELLED)

## Data Structure cần dùng:
- `ArrayList` cho danh sách items trong order
- Implement `Comparable<Order>` để dùng với PriorityQueue
